#### 跳表(Skip List)
跳表，对标的是平衡树和二分查找，是一种插入/删除/搜索都是O(log n) 的数据结构，跳表里面的元素始终是有序的。

只能用于元素有序的情况
用于解决链表搜索的问题，给有序的链表加速

优势：原理简单，容易实现，方便扩展，效率最高
