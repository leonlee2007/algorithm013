## 学习笔记

####哈希表（Hash table） 
  也叫散列表，是根据关键码值（Key value）而直接进行访问的数据结构
  通过关键码映射到表中一个位置来访问记录，以加快查找的速度
  这个映射函数叫作散列函数（Hash Function)
  存放记录的数组叫作哈希表（或散列表）
